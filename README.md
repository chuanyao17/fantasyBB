# Fantasy Basketball Assistant

一個基於 Yahoo Fantasy API 的籃球數據助手。

## 功能特點

- Yahoo OAuth 認證整合
- 獲取 NBA 聯盟資訊
- 獲取隊伍資料
- CORS 支援

## 技術棧

### 後端
- FastAPI
- Python 3.12
- Yahoo Fantasy API
- OAuth2

### 前端 (進行中)
- Next.js 13+
- TypeScript
- TailwindCSS

## 開始使用

### 前置需求
- Python 3.12+
- Node.js 18+
- Yahoo 帳號
